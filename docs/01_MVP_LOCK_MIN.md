# 01_MVP_LOCK_MIN.md

문서 목적: 현재 합의된 **최소판 최종 MVP 범위**를 잠근다.

## 1. 대상
- 부모 계정 1명
- 학생 프로필 1명: 수지

## 2. 목표
- 중1 우선의 회복형 수학 학습 앱
- 문제는 DB에서 공급
- 설명이 부족할 때만 API를 사용

## 3. 문제 공급 방식
- 아빠가 앱 밖에서 ChatGPT로 문제를 만든다
- 문제를 Excel로 정리한다
- 앱에 업로드한다
- 앱은 업로드된 문제를 DB에 저장한다
- 학생은 DB에 저장된 문제를 푼다

## 4. 학생 쪽 최소 기능
- 체크인
- 오늘의 추천 문제
- 문제 풀이
- 힌트
- 기본 해설
- 추가 설명 요청
- 오답 복습

## 5. 부모 쪽 최소 기능
- 오늘 학습 현황
- 약한 단원 TOP 3
- 실수 유형 TOP 3
- 회복 지표
- 오늘 아빠가 도와줄 포인트
- 문제 Excel 업로드
- 업로드된 문제 목록 보기
- 공개 여부 관리

## 6. API 사용 원칙
- 문제 생성에는 사용하지 않음
- 문제 검수에도 MVP에서는 사용하지 않음
- 수지가 설명을 모르거나 추가 설명이 필요할 때만 사용

## 7. 지금 하지 않는 것
- 학생 런타임 실시간 문제 생성
- 중2~고3 실제 콘텐츠 입력
- 강화학습 추천
- 다중 학생 지원
- 복잡한 운영 보고 체계
